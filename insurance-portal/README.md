# MiqorAI Insurance Portal

Web workspace for insurers: savings analytics, fraud detection, member management, and reporting.

## Development

```bash
npm install
npm run dev
```

Runs at **http://localhost:8081**.

## Build

```bash
npm run build
npm run preview
```

## Docker

```bash
docker build -t MiqorAI-insurance-portal .
docker run -p 8081:80 MiqorAI-insurance-portal
```
